# movie-websites-database
该仓库致力于构建一个影视类网站的基础信息数据库，用于归档公开可访问的电影网站及相关平台资料。收录内容主要包括站点名称、链接地址、站点定位说明以及基础分类信息，方便对影视站点进行系统化管理与查询。本项目不存储、不解析、不传播任何影视内容，仅作为信息记录与研究样本存在。所有数据均整理自公开网络环境，不对第三方网站的内容合法性、服务质量或安全性作任何保证或评价。仓库内容仅供资料查询和技术研究参考，不构成任何形式的推荐。
